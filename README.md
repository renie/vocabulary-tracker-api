# Base NodeJS API + MongoDB project 

## Install deps
```
npm i
```

## Setup your `.env`
Open your .env and setup everything

## Dev start
```
npm run dev
```

## Start
```
npm start
```

## If you want it to work as a serverless application
Install Serverless CLI and login
```
npm i -g serverless
serverless login
```

Deploy the app
```
serverless deploy
```
